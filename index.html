Proyek Saya

Aplikasi Todo List

Aplikasi sederhana untuk mencatat kegiatan sehari-hari.

Proyek ini dibangun dengan HTML, CSS, dan JavaScript murni. Fitur: tambah tugas, tandai selesai, hapus tugas, dan menyimpan data di localStorage.

Kalkulator Sederhana

Kalkulator dengan operasi dasar matematika.

Kalkulator ini mendukung penjumlahan, pengurangan, perkalian, pembagian, dan reset. Dibuat dengan CSS Grid untuk tata letak tombol.

Website Cuaca

Menampilkan cuaca berdasarkan nama kota.

Menggunakan API OpenWeatherMap. Pengguna bisa mencari cuaca di kota mana pun, dan data ditampilkan secara dinamis.

Gallery Foto Responsif

Galeri foto dengan efek lightbox.

Galeri ini menggunakan CSS Grid untuk layout responsif. Saat gambar diklik, akan muncul modal dengan gambar yang lebih besar.

Form Validasi Real-time

Form dengan validasi langsung saat mengetik.

Form ini memvalidasi nama, email, dan pesan secara langsung. Email dicek formatnya, dan pesan minimal 20 karakter. Pesan error muncul di bawah field.
